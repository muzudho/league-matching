「　永遠に上手くいく証明はできるの？」  

```plaintext
ラウンド番号を序数 r とする
参加人数を yy とする。yy は偶数とする
参加者番号は基数 y とする

リーグでの対戦相手番号は

(y+yy+r)%yy    if 自分の参加者番号が偶数
(y+yy-r)%yy    if 自分の参加者番号が奇数
```
